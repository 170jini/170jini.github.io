Daum 검색 API는 포털 사이트 Daum에서 방대한 웹 문서, 동영상, 이미지, 블로그, 책, 카페를 검색하는 기능을 제공합니다. 검색 결과는 JSON 객체로 전달돼 서비스에서 자유롭게 출력하거나 활용할 수 있습니다.     
## 웹문서 검색    
다음 검색 서비스에서 질의어로 웹 문서를 검색합니다. 원하는 검색어와 함께 결과 형식 파라미터를 선택적으로 추가할 수 있습니다. 응답 바디는 meta, documents로 구성된 JSON 객체입니다.    
{% highlight kotlin %}
/**
 * 키워드로 Daum 포털 사이트에서 관련 웹 문서 검색
 * @param query - 검색을 원하는 질의어 (Required)
 * @param sort - 결과 문서 정렬 방식, accuracy(정확도순) 또는 recency(최신순), 기본 값 accuracy
 * @param page - 결과 페이지 번호, 1~50 사이의 값, 기본 값 1
 * @param size - 한 페이지에 보여질 문서 수, 1~50 사이의 값, 기본 값 10
 */
fun webDocument(query: String, sort: String = "accuracy", page: Int = 1, size: Int = 10) {
    GlobalScope.launch() {
        val URI_BASE = "https://dapi.kakao.com/v2/search/web?"
        val obj = URL(
            URI_BASE.plus("query=").plus(query)
                .plus("&sort=").plus(sort)
                .plus("&page=").plus(page)
                .plus("&size=").plus(size)
        )
        val huc = obj.openConnection() as HttpURLConnection
        huc.requestMethod = "GET"
        huc.addRequestProperty("Authorization", "KakaoAK " + {REST_API_KEY})
        Log.d(TAG, "responseCode: ${huc.responseCode}")
        if (huc.responseCode == HttpURLConnection.HTTP_OK) {
            val streamReader = InputStreamReader(huc.inputStream)
            val buffered = BufferedReader(streamReader)
            val content = StringBuilder()
            while (true) {
                val data = buffered.readLine() ?: break
                content.append(data)
            }
            buffered.close()
            huc.disconnect()
            Log.d(TAG, "${content}")
        }
    }
}

## 이미지 검색
키워드로 Daum 포털 사이트에서 관련 이미지 검색    
* Request
{% highlight java %}

    GET /v2/search/image HTTP/1.1
    Host: dapi.kakao.com
    Authorization: KakaoAK {REST_API_KEY}

    query (String): 검색을 원하는 질의어 - Required
    sort (String): 결과 문서 정렬 방식, accuracy(정확도순) 또는 recency(최신순), 기본 값 accuracy
    page (Integer): 결과 페이지 번호, 1~50 사이의 값, 기본 값 1
    size (Integer): 한 페이지에 보여질 문서 수, 1~80 사이의 값, 기본 값 80

{% endhighlight %}
* Response
{% highlight java %}

    [documents] {
        collection (String): 컬렉션,
        thumbnail_url (String): 미리보기 이미지 URL,
        image_url (String): 이미지 URL,
        width (Integer): 이미지의 가로 길이,
        height (Integer): 이미지의 세로 길이,
        display_sitename (String): 출처,
        doc_url (String): 문서 URL,
        datetime (Datetime): 문서 작성시간, ISO 8601, [YYYY]-[MM]-[DD]T[hh]:[mm]:[ss].000+[tz]
    },
    meta {
        total_count (Integer): 검색된 문서 수,
        pageable_count (Integer): total_count 중 노출 가능 문서 수,
        is_end (Boolean): 현재 페이지가 마지막 페이지인지 여부, 값이 false면 page를 증가시켜 다음 페이지를 요청할 수 있음
    }

{% endhighlight %}

## 블로그 검색
키워드로 Daum 블로그 서비스에서 관련 게시물 검색    
* Request
{% highlight java %}

    GET /v2/search/blog HTTP/1.1
    Host: dapi.kakao.com
    Authorization: KakaoAK {REST_API_KEY}

    query (String): 검색을 원하는 질의어, 특정 블로그 글만 검색하고 싶은 경우, 블로그 url과 검색어를 공백(' ') 구분자로 넣을 수 있음 - Required
    sort (String): 결과 문서 정렬 방식, accuracy(정확도순) 또는 recency(최신순), 기본 값 accuracy
    page (Integer): 결과 페이지 번호, 1~50 사이의 값, 기본 값 1
    size (Integer): 한 페이지에 보여질 문서 수, 1~50 사이의 값, 기본 값 10

{% endhighlight %}
* Response
{% highlight java %}

    [documents] {
        title (String): 블로그 글 제목,
        contents (String): 블로그 글 요약,
        url (String): 블로그 글 URL,
        blogname (String): 블로그의 이름,
        thumbnail (String): 검색 시스템에서 추출한 대표 미리보기 이미지 URL, 미리보기 크기 및 화질은 변경될 수 있음,
        datetime (Datetime): 블로그 글 작성시간, ISO 8601, [YYYY]-[MM]-[DD]T[hh]:[mm]:[ss].000+[tz]
    },
    meta {
        total_count (Integer): 검색된 문서 수,
        pageable_count (Integer): total_count 중 노출 가능 문서 수,
        is_end (Boolean): 현재 페이지가 마지막 페이지인지 여부, 값이 false면 page를 증가시켜 다음 페이지를 요청할 수 있음
    }
